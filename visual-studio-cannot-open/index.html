Visual Studio無法開啟

發表於 2016-05-25 更新於 2025-05-14 分類於 工具 閱讀次數： Disqus：

要開發的時候，最鬱悶的我想應該就是熱血已經起來，準備打開IDE好好的大展身手一下
結果IDE給你停在啟動畫面，怎麼樣他就是不給你進去… …

其實事情是這樣的，使用了enterprice的試用評估，時間到了之後改回personal版本，後來幾天用的都挺好
某天突然想練一下window form，於是開啟專案之後，出現一個錯誤，接著vs當掉自動關閉

他就再也開不起來了，其實以前碰過n次這種事情，每次都用重灌解決，耗時耗力又折磨人
一直努力下關鍵字，都不夠精準，無奈之下隨手打個【visual studio 無法啟動】
發現了一篇文章有提到用命令列下參數開啟visual studio
(傳送門：[Visual Studio] Microsoft Visual Studio 2013 已停止工作　解決方案)

發現指令當然就是試一下，反正都開不起來了情況也不會更糟
令人訝異的是，打第一個指令，進入安全模式就很快速的開起來了
讓我多了信心，確定問題應該就是使用者的一些設定之類的吧~

又找到了Bruce的一篇文章，發現了一個關鍵指令，看起來就是【清除使用者設定】
(傳送門：學習使用DEVENV.EXE幫VISUAL STUDIO偵錯)

直接執行後，VS2015順利開啟，比重灌好很多了
