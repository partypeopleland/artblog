使用 Hexo 建立個人部落格

發表於 2019-04-20 更新於 2025-05-14 分類於 工具 閱讀次數： Disqus：

看了好幾篇 hexo 的文章，加上自己也已經很習慣使用 VScode 作為日常開發
尤其是 markdown 文件的撰寫便利，想想還是自己架個小小的 blog 紀錄日常心得吧

為甚麼用 Hexo

對我而言 hexo 有幾項優點：

  1. 安裝快速
  2. 靜態 HTML：不需後端伺服器，所以可以架在自己的 NAS 上面
  3. 更換佈景主題方便，且資源眾多，雖說重點只是記錄知識，但是偶爾換換主題也是挺賞心悅目的一件事
  4. 採用 markdown 撰寫，而且 VSCode 的支援度頗高
  5. 容易客製調整，基本上就是 HTML 與 JavaScript，如果哪天想要改了也很方便
  6. 有多種佈署外掛，包含 github、heroku 等空間，也有支援 FTP/SFTP 的方式來佈署

    佈署相關可參照官方文件：https://hexo.io/docs/deployment.html
